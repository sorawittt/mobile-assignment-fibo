## Getting Started

### 1. Install Dependencies
Run the following command to install all required dependencies:
```bash
flutter pub get
```

### 2. Generate Code (Only test)
```bash
dart run build_runner build --delete-conflicting-outputs   
```

### 3. VS Code Configuration
```json
{

    "configurations": [

            "name": "fibo",
            "request": "launch",
            "type": "dart"
        },
    ]
}
```
